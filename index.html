How to install

ReMind is distributed here, not on the Play Store, so Android asks you to approve the install the first time. It takes about a minute:

  1. Tap Download ReMind above. If your browser warns that the file "might be harmful", choose Download anyway. This is Android's standard notice for any app installed outside the Play Store.
  2. When the download finishes, tap the file in the notification shade, or open it from your Files app (usually in the Downloads folder).
  3. Android shows: "For your security, your phone is not allowed to install unknown apps from this source." Tap Settings.
  4. Turn on Allow from this source for your browser, then go back. You can turn this off again after installing.
  5. Tap Install on the ReMind prompt, then Open.
  6. Allow notifications when ReMind asks on first launch. Reminders can't alert you without it.
